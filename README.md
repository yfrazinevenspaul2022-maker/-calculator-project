# Python Calculator 🧮

## Description
A command-line calculator built with Python as part of the **100 Days of Code** challenge (Day 10).
Supports chained calculations without restarting the program.

## Features
- Addition ➕
- Subtraction ➖
- Multiplication ✖️
- Division ➗
- Chain calculations with previous result
- Option to restart or continue

## How to Use
1. Run main.py
2. Enter your first number
3. Pick an operation (+, -, *, /)
4. Enter your next number
5. Type Y to continue calculating with the result
6. Type N to restart with a new number

## Example

    What's your number? 10
    +
    -
    *
    /
    Pick a operation: +
    What's your next number? 5
    10.0 + 5.0 = 15.0
    type Y to continue calculating with 15.0 or N to restart: Y
    +
    -
    *
    /
    Pick a operation: *
    What's your next number? 2
    15.0 * 2.0 = 30.0
    type Y to continue calculating with 30.0 or N to restart: N
